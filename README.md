# Python Condition Worksheet

This repository contains my solutions for a Python condition worksheet completed during my internship preparation/training.

The goal of this work is to practice Python fundamentals, especially conditional statements, user input, comparison operators, logical operators, and basic problem-solving.

## Topics Covered

* Python input and output
* Variables and data types
* If / elif / else statements
* Comparison operators
* Logical conditions
* Basic arithmetic operations
* Simple classification problems
* Loop usage in FizzBuzz
* Clean and readable Python code

## Exercises

### Exercise 1: Number Comparison

A program that takes two numbers as input and checks whether the first number is greater than, smaller than, or equal to the second number.

### Exercise 2: Tax Calculator

A program that calculates income tax based on the user's income:

* Income less than 50,000: 10% tax
* Income between 50,000 and 100,000: 20% tax
* Income greater than 100,000: 30% tax

### Exercise 3: BMI Calculator

A program that calculates the Body Mass Index using weight and height, then classifies the result as:

* Underweight
* Normal
* Overweight
* Obese

### Exercise 4: Sorting Numbers

A program that takes three numbers as input and prints them in ascending order.

### Exercise 5: Day of the Week

A program that takes a number from 1 to 7 and prints the corresponding day of the week.

### Exercise 6: Age Group Classifier

A program that takes a person's age and classifies them as:

* Child
* Teen
* Adult
* Senior

### Exercise 7: Vowel or Consonant Checker

A program that reads a letter from the user and checks whether it is:

* A vowel
* A consonant
* The letter `y`, which can sometimes be a vowel and sometimes a consonant

### Problem: FizzBuzz

A program that prints numbers from 1 to N.

Rules:

* Multiples of 3 print `Fizz`
* Multiples of 5 print `Buzz`
* Multiples of both 3 and 5 print `FizzBuzz`
* Other numbers print the number itself

## How to Run

Make sure Python is installed on your machine.

This project uses one main Python file: `task.py`.

To run the program, use:

```bash
python task.py
```

After running the file, a menu will appear:

```text
Which exercise do you want to run?
1. Exercise 1 - Number Comparison
2. Exercise 2 - Tax Calculator
3. Exercise 3 - BMI Calculator
4. Exercise 4 - Sorting Numbers
5. Exercise 5 - Day of the Week
6. Exercise 6 - Age Group Classifier
7. Exercise 7 - Vowel or Consonant
8. Exercise 8 - FizzBuzz
9. Exit
```

Enter the number of the exercise you want to run.

Example:

```text
Enter your choice: 3
```

This will run the BMI Calculator exercise.
